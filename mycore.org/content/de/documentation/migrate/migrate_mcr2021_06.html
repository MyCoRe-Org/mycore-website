Migrationskommandos einbinden

Java 16

CLI und ältere Tomcat-Versionen funktionieren nicht mit Java 16 Damit das wieder funktioniert muss ein aktueller Tomcat genutzt werden und zusätzliche Parameter in der CLI ergänzt werden. Siehe dazu z.B. https://mycore.atlassian.net/browse/MIR-1031 bzw. https://github.com/MyCoRe-Org/mir/commit/13e014cb0359d5f618924b5823c7ce6087c918ab
