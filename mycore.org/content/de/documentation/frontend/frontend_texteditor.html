Texteditor in MyCoRe

Der Texteditor in MyCoRe bietet eine einfache und schnelle Möglichkeit Objekte, Derivate, Klassifikation und Textdokumente innerhalb von Derivaten zu ändern oder zu betrachten.

Er bietet Syntax-Highlighting für die meisten Textformate und kann Zeilennummern anzeigen.

Integration in die Webseite

Typ URL
Objekt XML /modules/webtools/texteditor/objects/{mcr_object_id}
Derivat XML /modules/webtools/texteditor/objects/{mcr_object_id}/derivates/{mcr_derivate_id}
Textdokument /modules/webtools/texteditor/objects/{mcr_object_id}/derivates/{mcr_derivate_id}/contents/{path_to_file}
Klassifikation /modules/webtools/texteditor/classifications/{classification_id}

Weitere Informationen

Der Texteditor verwendet im Backend die Rest v2 API. Aus diesem Grund ist es theoretisch möglich, auf Nicht-Textdateien (wie z.B. TIFF) zuzugreifen. Der Texteditor kann diese natürlich nicht korrekt darstellen. Der Aufrufer (Link-Ersteller) hat dementsprechend dafür Sorge zu tragen, dass der Texteditor nur mit Textdokumenten aufgerufen wird.

Syntax Highlighting

Aktuell unterstützt der Texteditor folgende Formate:

HTML, XML, SVG, MathML, SSML, Atom, RSS, Markup, Markdown, CSV, YAML

Weitere Formate können bei Bedarf integriert werden. Siehe Link.
