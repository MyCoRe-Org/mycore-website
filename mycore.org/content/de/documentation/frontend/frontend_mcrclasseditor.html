Der Klassifikations-Browser

Der Klassifkations-Editor biete die Möglichkeit im System gespeicherten Klassifikationen direkt zu bearbeiten. Er kann über einen Template-Aufruf direkt in eine externe Web-Seiten eingebunden werden.
