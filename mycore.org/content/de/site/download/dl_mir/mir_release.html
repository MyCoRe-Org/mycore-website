Download der aktuellen SNAPSHOT-Version

Hier kann die aktuelle LTS-Version der Web-Anwendung mir-webapp ($VERSION) (speichern als mir.war) heruntergeladen werden. Dieses kann mit dem Context 'mir' direkt in einen Servlet Container ausgeliefert (deployed) werden.

Für administrative Zwecke kann zusätzlich eine Kommandozeile (MIR-CLI) im Format zip oder tar.gz heruntergeladen werden:

  • ZIP-Archiv ($VERSION) (speichern als mir-cli.zip)
  • TAR.GZ-Archiv ($VERSION) (speichern als mir-cli.tar.gz)

Die Neuerungen seit dem letzten Release (2025.06) und notwendigen Schritte für die Aktualisierung der Anwendung finden Sie in der Migrationsanleitung "MIR LTS 2025.06 nach 2025.12".
