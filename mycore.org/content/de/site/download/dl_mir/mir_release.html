Hier kann das aktuelle MIR-LTS-Paket (speichern als mir.war) heruntergeladen werden. Dieses kann mit dem Context 'mir' direkt in einen Servlet Container ausgeliefert (deployed) werden.

Für administrative Zwecke kann zusätzlich eine MIR-CLI im Format zip oder tar.gz heruntergeladen werden:

  • zip archive (speichern als mir-cli.zip)
  • tar.gz archive (speichern als mir-cli.tar.gz)

Die Liste der Änderungen ist im Changelog zu finden.
